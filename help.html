Controls

Use keys W, A, S, D to move the player. Q creates a new world, and Z-X switch the tile type.

Mouse: left-click on a empty space to set a tile in it (the default tile is air, so try using X before.) Right-clicking a tile will clean it.
Middle-clicking a tile will sample it.
